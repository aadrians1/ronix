Copy the files to a directory of your choice.
Please run first the installer: SWUPINST.EXE which generates an
INI-File.
After this you can run the Software-Update-Program:

 S25_0501.EXE

Without a parameter it starts the software-update.

You get help with :                          S25_0501    -h
information about the SW is available with:  S25_0501    -i
